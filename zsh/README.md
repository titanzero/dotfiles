# 🐚 Zsh

> Shell config with [Oh My Zsh](https://ohmyz.sh/) and [Powerlevel10k](https://github.com/romkatv/powerlevel10k).

## Files

| File        | Description                        |
|-------------|------------------------------------|
| `.zshrc`    | Main shell config, aliases, paths  |
| `.p10k.zsh` | Powerlevel10k prompt theme         |

## Stow

```bash
stow -t ~ zsh
```
